# ClaudePBI
Details of how I am connecting Claude to Power BI Via MPC

Following this guide: https://beyondtheanalytics.com/blog/claude-ai-power-bi-mcp-setup-guide-2026

(1) Install Node.js: https://nodejs.org/en/download
    - This also installed Chocolatey, Python, Visual Studio, VS Build Tools + VC Redistributable

(2) Install Claude Desktop: https://claude.com/download

(3) Install Microsoft MCP Server: https://learn.microsoft.com/en-us/power-bi/developer/mcp/remote-mcp-server-get-started
